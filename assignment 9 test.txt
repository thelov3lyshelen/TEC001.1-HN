Write a function that implements the Caesar cipher technique. In particular:
It should have 3 parameters: the file name, how many positions to shift, and the shifting direction (left/right).
The function will open the file, and for every character (excluding numbers), replace it with a corresponding Caesar character. The general formula is as follows:
For uppercase chars: y = (x ± n - 65) mod 26 + 65
For lowercase chars: y = (x ± n - 97) mod 26 + 97, where x is the ASCII value of a character, n is the # of positions to shift, and y is the new ASCII value +/- depends on the shifting direction (right: +, left: -).
For example, if you shift a character 3 positions to the right, then A becomes D, G becomes J, and Z becomes C.
(Hint: to get the ASCII value of a character, you can use ord(c).)
